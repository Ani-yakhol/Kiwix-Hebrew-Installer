## מתקין קוויקס בעברית
זהו מתקין עצמאי מותאם לעברית עבור#  Kiwix Desktop, ואינו מופץ באופן רשמי על ידי ארגון Kiwix
# מדריך שימוש – מעטפת ההתקנה של Kiwix Desktop (Inno Setup)

## מבנה התיקיות שבחבילה

```
Source\              קבצי תוכנת Kiwix (כפי שחולצו מקובץ ה-ZIP שהעלית, כולל kiwix-desktop.exe ו-vc_redist.x64.exe)
Icons\kiwix.ico       האייקון האמיתי של התוכנה, חולץ מתוך kiwix-desktop.exe (כולל כל הגדלים: 16-256 פיקסלים)
Installer\setup.iss   סקריפט ה-Inno Setup הראשי
Installer\Languages\Hebrew.isl   קובץ תרגום עברית + RTL עבור Inno Setup
Installer\Output\     לכאן ייוצא קובץ ה-Setup.exe הסופי לאחר קומפילציה
README.md              המדריך הזה
```

## איך בונים את ההתקנה
1. התקן את **Inno Setup 6** (לא פחות) מהאתר הרשמי jrsoftware.org.
2. פתח את `Installer\setup.iss` בעזרת Inno Setup Compiler (ISCC / IDE).
3. לחץ Build/Compile. הקובץ הסופי ייווצר בתיקיית `Installer\Output`.
4. ודא ש-`Hebrew.isl` נמצא בתיקיית `Languages` היחסית לסקריפט (זה כבר מוגדר נכון בחבילה).

## הפונקציות שהוטמעו בקובץ ה-ISS ולמה הן משמשות

| פונקציה | הסבר קצר |
|---|---|
| **[Languages] + Hebrew.isl** | קובע עברית כשפת ההתקנה, עם `RightToLeft=yes` ב-`[LangOptions]` שבתוך קובץ ה-ISL. זה גורם ל-Inno Setup ליישר את **כל** הטקסטים, הכפתורים, תיבות הסימון ותיבות הרשימה מימין לשמאל אוטומטית בכל עמודי האשף הסטנדרטיים. |
| **AppName / UninstallDisplayName** | קובעים ששם התוכנה שיופיע גם בכותרת האשף וגם ברשימת "הסרת תוכניות" בלוח הבקרה יהיה **"קוויקס – קורא קבצי זים"**. |
| **[Icons]** | יוצר את קיצורי הדרך: בתיקיית תפריט ההתחל (תמיד) ועל שולחן העבודה (רק אם המשתמש סימן את התיבה המתאימה ב-Tasks). קיצור ההסרה נוצר גם הוא אוטומטית. |
| **[Tasks] → desktopicon** | תיבת סימון אופציונלית (לא מסומנת כברירת מחדל) ליצירת קיצור על שולחן העבודה. |
| **[Tasks] → assoczim** | תיבת סימון אופציונלית (מסומנת כברירת מחדל) לשיוך קבצי `.zim` לפתיחה אוטומטית עם kiwix-desktop.exe. |
| **[Registry] (שיוך ZIM)** | רושם את הסיומת `.zim`, את סמל הקובץ ואת פקודת הפתיחה במפתחות `HKA` (=HKCU בהתקנה למשתמש נוכחי, או HKLM בהתקנת מנהל-מערכת), וגם רושם את התוכנה תחת `RegisteredApplications` כדי שתופיע בתפריט "פתח באמצעות" ו-"תוכניות ברירת מחדל" של Windows. |
| **[Registry] (מידע כללי)** | שומר את נתיב ההתקנה והגרסה במפתח רישום ייעודי לתוכנה, לשימוש עתידי (עדכונים, איתור התקנה קיימת וכו'). |
| **VCRedistNeedsInstall (Pascal)** | פונקציה שבודקת את מפתח הרישום הרשמי של מיקרוסופט (`...VC\Runtimes\X64\Installed`) כדי לדעת אם ה-Visual C++ Redistributable כבר מותקן במחשב. אם כן – לא תתבצע התקנה נוספת שלו. |
| **[Run] → vc_redist.x64.exe** | אם `VCRedistNeedsInstall` מחזירה True, ההתקנה מריצה את `vc_redist.x64.exe /install /quiet /norestart` – **התקנה שקטה לחלוטין**, ללא כל חלונית שנראית למשתמש. |
| **SetupIconFile** | קובע את האייקון של **קובץ ה-Setup.exe עצמו** (מה שרואים בסייר הקבצים לפני ההרצה). |
| **WizardSmallImageFile** | קובע את האייקון הקטן שמופיע **בתוך חלונית ההתקנה עצמה** (בפינה העליונה של כל עמוד באשף). |

## איך משתמשים באייקונים
- קובץ `Icons\kiwix.ico` הוא קובץ ICO אמיתי (רב-רזולוציה: 16x16 ועד 256x256), שחולץ ישירות מתוך המשאבים (Resources) של `kiwix-desktop.exe` – כך שהוא זהה לחלוטין לאייקון המקורי של התוכנה.
- בקובץ ה-ISS הוא כבר מקושר בהגדרה (`SetupIconFile``), אין צורך לגעת בהן.
-אייקון המוצג בחלונית האשף עצמה (בפינה השמאלית העליונה, גודל קטן) 
WizardSmallImageFile={#IconsPath}\kiwix_small.bmp
- אם תרצה גם תמונת רקע גדולה יותר לעמוד הפתיחה/סיום של האשף (WizardImageFile, מומלץ 164×314 פיקסלים בערך), פשוט צור קובץ BMP בגודל המתאים, שים אותו בתיקיית `Icons`, והוסף שורה כמו שמצוין בהערה שבתוך ה-ISS.
- קיצורי הדרך ב-[Icons] כבר מפנים ל-`IconFilename: "{app}\kiwix-desktop.exe"` – כלומר קיצור הדרך "שואב" את האייקון ישירות מתוך קובץ ה-EXE המותקן, אין צורך בקובץ ICO נפרד עבורם.

## איך להפיץ גרסה **בלי** vc_redist.x64.exe (הפצה קלה יותר)
קובץ ה-vc_redist.x64.exe (כ-25 מ"ב) כלול כברירת מחדל בתוך חבילת ההתקנה. אם ברצונך להפיץ גרסה שאינה כוללת אותו כלל (למשל להורדה דרך האינטרנט כדי לחסוך משקל, או כי אתה בטוח שהוא כבר מותקן אצל המשתמשים שלך):

**אפשרות א' – קומפילציה עם דגל מיוחד (מומלץ, לא דורש עריכת קובץ):**
הרץ את הקומפיילר משורת הפקודה עם ההגדרה `SKIP_VCREDIST_EMBED`:
```
ISCC.exe /DSKIP_VCREDIST_EMBED Installer\setup.iss
```
כאשר דגל זה מועבר, הבלוק המסומן ב-`#ifdef IncludeVCRedist` בקובץ ה-ISS לא ייכלל בקומפילציה, kiwix-desktop.exe קובץ ה-vc_redist לא יועתק לחבילה, וגם פקודת ההרצה השקטה שלו לא תתבצע.

**אפשרות ב' – הסרה ידנית של הקובץ:**
פשוט מחק את `vc_redist.x64.exe` מתוך תיקיית `Source` לפני הקומפילציה. הבלוק `#ifdef IncludeVCRedist` עדיין ינסה לכלול את הקובץ (ויכשל אם הוא לא קיים), ולכן אם בוחרים בדרך זו יש גם למחוק/להעיר (comment out) את שתי השורות הרלוונטיות תחת `[Files]` ו-`[Run]` שמתייחסות ל-vc_redist.

> מומלץ להשתמש באפשרות א' (`/DSKIP_VCREDIST_EMBED`) – היא נקייה יותר ולא דורשת עריכת קוד.

## הערות נוספות
- ההתקנה דורשת הרשאות מנהל מערכת (`PrivilegesRequired=admin`) כדי לאפשר רישום תקין ב-Registry ושיוך קבצים לכלל המשתמשים. אם תרצה לאפשר גם התקנה ללא הרשאות מנהל (למשתמש הנוכחי בלבד), אפשר לשנות ל-`PrivilegesRequiredOverridesAllowed=commandline` (כבר קיים) ולהתאים את `PrivilegesRequired=lowest`.
- כל הטקסטים בממשק (כפתורים, תוויות, הודעות שגיאה, עמוד "סיום" וכו') מתורגמים באופן מלא בקובץ `Hebrew.isl` – כולל התאמת RTL אמיתית לכל האלמנטים הגרפיים הסטנדרטיים של Inno Setup.
- שם התוכנה שמופיע בלוח הבקרה תחת "הוספה/הסרה של תוכניות" הוא **"קוויקס – קורא קבצי זים"**, כפי שהוגדר ב-`UninstallDisplayName`.
